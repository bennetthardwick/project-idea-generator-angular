what's it for

This Angular 5 app was made for the CODE Network showcase night. It was made in just a few hours before the event because I was upset I didn't have anything to show.

making ideas

It's super easy! Just click the generate button.

adding data

I use this d3.js application to define the grammar for the app. Words that are prefixed with an underscore '_' are placeholders for random words. For example, _noun would be substituted with a random noun. Likewise, _adjective would be substituted by a random adjective. The graph must also start at _start and end at _end. Download and upload this JSON to the application to see and edit the current grammar that the application uses. If I feel like it I'll probably end up making this a feature of the generator.

If you'd like to bulk add dictionary data to the app, have a look at this JSON.

After you have your JSON files ready, hit up the settings page and link them from a website that doens't use CORS, like gist.github.com.

how it works

Each sentence is essentially a journey across a directed graph which controls the grammar. When the app reaches a node that has more than one children, it chooses a path randomly, with equal probability. This continues until the app reaches the end of the graph and the random sentence is generated. Check out the source code on github.
